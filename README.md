# 🌤️ weather-news-bot\n\n**Telegram bot para alertas en tiempo real de clima y noticias globales.**\n\n![Python](https://img.shields.io/badge/Python-3.13+-blue?logo=python)\n![License](https://img.shields.io/badge/License-MIT-green)\n![Status](https://img.shields.io/badge/Status-Active-brightgreen)\n\n---\n\n## 🌐 Descripción\n\nEste bot de Telegram combina dos servicios públicos sin API key:\n- **Clima**: Usa [Open-Meteo](https://open-meteo.com/) para datos meteorológicos en tiempo real.\n- **Noticias**: RSS de [BBC World News](https://www.bbc.com/news/world) para actualidad global.\n\nDiseñado para ser **ligero, rápido y funcional en dispositivos móviles** (Termux/Android).\n\n---\n\n## ⚙️ Requisitos\n\n- Python 3.13+\n- `requests` (ya incluido en Termux por defecto)\n- Token de bot de Telegram (creado con [@BotFather](https://t.me/BotFather))\n\n---\n\n## 🚀 Instalación y Uso\n\n### 1. Clonar y ejecutar\n```bash\ngit clone https://github.com/aroldanruda-wq/weather-news-bot.git\ncd weather-news-bot\npython3 bot.py\n```\n\n### 2. Pruebas locales (sin token)\n```bash\npython3 bot.py\n```\nVerás una simulación de los comandos `/weather` y `/news`.\n\n### 3. Ejecutar con token real\n```bash\npython3 bot.py TU_TOKEN_AQUI\n```\n> **Nota**: Para producción completa, instala `python-telegram-bot`:\n> ```bash\n> pip install python-telegram-bot\n> ```\n\n---\n\n## 📡 Comandos Disponibles\n\n| Comando | Descripción | Ejemplo |\n|---------|-------------|---------|\n| `/weather` | Muestra temperatura, viento y dirección actual | `📍 Madrid: 22°C, Viento: 15 km/h` |\n| `/news` | Últimas 3 noticias globales con enlaces | `📰 BBC: [Título](enlace)` |\n\n---\n\n## 🌍 Soporte de Ubicación\n\nPara cambiar la ciudad, edita `bot.py` y modifica:\n```python\nCITY = \
